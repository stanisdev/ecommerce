Основные понятия:
+ Существуют главные категории, такие как, например: мужские одежда, женская, детская
+ У категории есть подкатегории, например: футболки, куртки
+ В меню есть пункт "Все", после чего мы попадаем на перечисление всех товаров подряд
  в данной категории.




Товар
+ Имеет цену
+ В наличии / или отсутсвует
+ Имееть скидку / или нет (в процентах)
+ Есть такие выпадающие пункты как "размер" или "цвет".
  Эти пункты отдельно указываются к товару в админке
+ Есть описание
+ "Shipping info"
+ Есть возможность добавить товар в "список желания" (wish_list)
+ Товар можно добавить в корзину (Cart)
+ При добавлении в корзину имеется возможность указать количество
+ У товара есть рейтинг (звездочки)
+ Товар относится как к минимум к одному бренду
+ Каждый товар включает в себя массив тегов. Теги имеются в bottom сайта





Фильтрация товаров
+ По бренду
+ По скидке
+ По типу



Сортировка
+ Best Selling
+ Alphabetically, A-Z
+ Alphabetically, Z-A
+ Price, low to high
+ Price, high to low
+ Date, new to old
+ Date, old to new



Страница "список желаний"
+ Перечисляем товар, который пользователь отметил как желаемый
